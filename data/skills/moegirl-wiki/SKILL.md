# Moegirl Wiki 查询

通过萌娘百科 (mzh.moegirl.org.cn) 查询 ACG 相关信息。

## 搜索方式

直接访问人物/角色页面：

```
https://mzh.moegirl.org.cn/{人物名}
```

示例：
- 角色查询: `https://mzh.moegirl.org.cn/时崎狂三`
- 声优查询: `https://mzh.moegirl.org.cn/花泽香菜`
- 作品查询: `https://mzh.moegirl.org.cn/进击的巨人`

## 工作流程

1. 用 `playwright__browser_navigate` 访问 `https://mzh.moegirl.org.cn/{关键词}`
2. 用 `playwright__browser_snapshot` 读取页面内容
3. 判断页面类型：
   - **人物页面**（直接命中）→ 提取关键信息回答用户
   - **消歧义页面**（多个同名条目）→ 选择最接近用户意图的条目，点击进入
   - **搜索结果页**（未命中，跳转到搜索）→ 从结果中选最相关的条目点击进入
4. 进入目标页面后，提取信息回答

## 关键词处理

- 中文名优先，日文名备选
- 角色全名优先，如"时崎狂三"而非"狂三"
- 声优搜索用中文译名，如"花泽香菜"而非"花澤香菜"
- 如果中文名没结果，尝试日文原名

## 信息提取要点

### 角色页面通常包含

- 基本信息（姓名、年龄、身高、生日等）
- 出处作品
- 声优/配音演员 (CV)
- 人物简介和经历
- 萌属性/角色标签

### 声优页面通常包含

- 基本信息（出生日期、出身地、所属事务所）
- 代表作品与角色列表
- 相关活动

## 注意事项

- 萌娘百科页面内容可能很长，优先读取页面顶部的信息框和简介部分
- 回答时用简洁的中文，不需要照搬整个页面
- 如果页面不存在（404），告知用户未找到相关信息

# Web Search

通过 `web_search` 和 `web_read` 工具进行网页搜索和信息提取。

## 工作流程

1. 使用 `web_search` 搜索关键词，获取结果列表
2. 如需详细内容，使用 `web_read` 读取具体页面
3. 汇总信息回复用户

## 搜索步骤

```
1. web_search(query="关键词") → 获取搜索结果（标题+URL+摘要）
2. (可选) web_read(url="具体页面URL") → 读取页面正文
3. 汇总关键信息回复用户
```

## 注意事项

- 搜索关键词应简洁精准，中文搜索用中文关键词
- 优先从搜索结果摘要中提取答案，避免不必要的页面读取
- 只有需要详细内容时才用 `web_read`
- 如果搜索结果不理想，尝试调整关键词重新搜索
- 汇总信息时注明来源
